💾 DATA LOADING IN PYTHON

CIFAR10:

  processor = CIFAR10Processor(config)
  train_loader, test_loader = processor.get_dataloaders(batch_size=128)

Fraud Detection:
  from src.preprocessing.fraud_processor import FraudProcessor
  processor = FraudProcessor(config)
  X, y = processor.preprocess(processor.load_data('data/credit_card_fraud/creditcard.csv'))

📋 KAGGLE SETUP (for fraud dataset)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install:
   pip install kaggle

2. Create API token:
   Go to: https://www.kaggle.com/settings/account
   Click: Create New API Token
   This downloads kaggle.json

3. Setup credentials:
   mkdir -p ~/.kaggle
   mv ~/Downloads/kaggle.json ~/.kaggle/
   chmod 600 ~/.kaggle/kaggle.json

4. Download:
   bash scripts/download_fraud_dataset.sh

⚠️  TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Dependencies fail:
  pip install --upgrade pip
  pip install -r requirements.txt

CIFAR10 download fails:
  Check internet connection
  Try again (downloads resume)

Kaggle auth fails:
  ls -la ~/.kaggle/kaggle.json
  Ensure file exists and has correct permissions

Space issues:
  df -h /
  Need ~250 MB free space

Verify existing setup:
  python3 src/preprocessing/download_datasets.py --verify-only
